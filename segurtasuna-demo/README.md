# Segurtasuna

Segurtasuna lantzerakoan hiru eraso mota ikusten ditugu, horietatik bi spyware bidezkoa eta phishing-a.

Estafagramek Instagrameko logina simulatzen du, eta bertan sartutakoa fitxategi batera bidali. Gero Instagrameko orrira berbideratzen du, baina firewallak ez du sartzen uzten, beraz, mobileko datuekin egin behar da demoa. Ikasleekin demoa egin behar bada, abixatu, ez dira fijatzen phishing orri bat dela.

Keyloggerra spyware bat da. Idatzitako guztia log fitxategian idazten du.
