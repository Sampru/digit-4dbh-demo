# Bitcoin

Bitcoinaren funtzionamendua azaltzen duen scripta.

Hashing-a azaltzerakoan, beti planteatzen dut erronka bat:
- Ikasleren batek 5 zero dituen sha256 hash bat aurkitzen badu (0x00000...), gaia baztertu eta gai horretan 10ekoa jarriko diet denei.
- Gela guztia honetan ibolukratzen da.
- Ez dute ezer lortzen, matematikoki ezinezkoa delako.
- Ordenagailuak segituan lortzen du, eta pikatu egiten dira.

Honekin gero bi irteera daude, Hashing ariketak egitea, edo Bitcoinek edo adimen artifizialak zergaitik gastatzen duen hainbeste hausnartzea.
